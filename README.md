# DP_Backend
## Environmet Setting
	Install PHP, Laravel, pg16 on your local
## .env File Setting
	create a .env file by yourself and add db content
	you can find this in our notes
## Dependany Install
	composer install
	npm install
## DB setting
	follow step order run create_database.sql in pg admin
## Migrate database
	php artisan migrate
## Start
	php artisan serve
